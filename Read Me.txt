# To check IP Reputation in web browser
1) Press windows + r
2) Enter cmd and run (keep cmd always open)
3) In command line, enter "cd /"
4) Then Enter "cd Alerts"
5) Key in "Open_Url xxx.xxx.xxx.xxx" (To check, replace xxx with IP)
6) For new IP, repeat step 5 only.
